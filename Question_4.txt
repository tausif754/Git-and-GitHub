Q4. Briefly explain what GitHub is and how it facilitates collaboration and version control also name some
alternatives to GitHub.
Answer
GitHub is a web-based platform for version control and collaboration that is widely used by 
software developers. It offers a centralized location for hosting and managing Git repositories,
 making it easier for individuals and teams to work together on code projects. 
 Here's how GitHub facilitates collaboration and version control:
 1.Repository Hosting:
 2.Collaboration Tools: 
 3.Version Control:
 4.Forks and Cloning: 
 5.Access Control:
 6.Integration with Continuous Integration (CI) Tools: 
 7.Discoverability:

 GitHub is known for its user-friendly interface, extensive documentation, and 
 a vast community of developers. It has both free and paid plans, depending on
the level of features and support required.

Alternative platforms to GitHub for code hosting and collaboration include:
1.GitLab: Similar to GitHub, GitLab provides Git repository hosting and collaboration tools.
 It offers self-hosted options and cloud-based services.

2.Bitbucket: Bitbucket is owned by Atlassian and offers Git and Mercurial repository hosting,
  along with features for collaboration and project management.
  
3.SourceForge: SourceForge is a platform for version control, bug tracking, and project management.
It has been around for a long time and hosts a wide variety of open-source projects.

4.GitKraken: GitKraken is a Git GUI client and collaboration platform that 
includes version control features and integrations.

5.Azure DevOps (formerly Visual Studio Team Services): Azure DevOps provides a suite of tools for
project management, version control, and CI/CD, and it integrates well with Microsoft technologies.
